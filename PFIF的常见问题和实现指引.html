      <h3 id="schema-location">
      What are the <code>xmlns:xsi</code> and
      <code>xsi:schemaLocation</code> attributes for?</h3>
       
      <p>
      These attributes are not required by the PFIF specification,
      but can help validation tools validate XML documents.
      For example,
      Altova&#39;s <a href="http://www.altova.com/products_ide.html">XML Spy</a>,
      an XML editor and validator, recognizes these attributes
      and can use them to validate a PFIF document against the PFIF XML Schema.
       
      <p>
      The <a href="http://zesty.ca/pfif/1.1/pfif-1.1-example.xml">example PFIF document</a>
      shows how to use these attributes
      to tell readers of a PFIF document
      where to find the XML Schema for the document.
      The <code>xmlns:xsi</code> attribute identifies the namespace
      for an XML Schema Instance,
      and the <code>xsi:schemaLocation</code>
      maps the PFIF namespace URI to the URL for the XML Schema document.
